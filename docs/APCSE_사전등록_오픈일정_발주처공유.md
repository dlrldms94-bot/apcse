# APCSE 사전등록 사이트 오픈 일정

**문서 목적:** 2026년 7월 27일(일) 사전등록 사이트 정식 오픈을 위한 진행 현황 및 향후 작업 일정 공유  
**작성 기준일:** 2026년 7월 9일  
**오픈 목표일:** 2026년 7월 27일  
**운영 주체:** 주식회사 온앤미  
**행사 브랜드:** APCSE

---

## 1. 프로젝트 개요

APCSE 행사 사전등록을 위한 웹사이트입니다. 신청자 유형에 따라 등록 절차가 다릅니다.

| 구분 | 등록 절차 | 결제 |
|------|-----------|------|
| **내국인** | 정보 입력 → 즉시 등록 완료 | 없음 |
| **외국인** | 정보 입력 → PayPal 결제 → 등록 완료 | PayPal (USD) |

등록 완료 후 신청자는 **마이페이지**에서 비밀번호로 로그인하여 등록 정보를 조회·수정할 수 있습니다.  
운영 담당자는 **관리자 로그 화면**에서 등록·결제·로그인 시도 등의 기록을 확인할 수 있습니다.

## 4. 오픈 전 필수 작업 일정

아래 일정은 7월 27일 정상 오픈을 기준으로 수립한 권장 일정입니다.

### 4.1 1단계: 즉시 착수 (7월 9일 ~ 7월 11일)

**목표:** 결제 테스트 완료 및 실결제 심사 신청

| 번호 | 작업 | 담당 | 비고 |
|------|------|------|------|
| 1 | Sandbox(테스트) 결제 1회 이상 완료 | 개발·운영 | 등록 → 결제 → 완료 → 마이페이지 → 관리자 로그 확인 |
| 2 | **PayPal 실결제(Live) 심사 신청** | 운영(온앤미) | **가장 시급** — 심사에 며칠~2주 소요 가능 |
| 3 | 사업자 PayPal 계정에 **은행계좌 연결** | 운영(온앤미) | 실결제 정산에 필수 |
| 4 | **등록비 금액·통화 최종 확정** | 발주처·운영 | 현재 기본값 USD 100 (변경 시 서버 설정 반영 필요) |
| 5 | **환불 정책 확정** | 발주처 | 취소·환불 시 PayPal 수동 환불 절차 안내 필요 |

#### Sandbox 결제 테스트 확인 항목

```
□ 외국인 등록 정보 입력
□ PayPal 테스트 결제 완료
□ 등록 완료 화면 표시
□ 마이페이지에서 결제 완료(PAID) 상태 확인
□ 관리자 로그에서 결제 성공 기록 확인
```

> **중요:** 7월 27일 실제 결제를 받으려면 PayPal Live 심사가 **이번 주 안에** 신청되어야 합니다. 심사 지연 시 오픈 일정에 영향을 줄 수 있습니다.
